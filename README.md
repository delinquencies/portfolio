# Nic Gibson — Portfolio

Static portfolio site: open `index.html` in a browser or serve the folder with any static server.

## GitHub

After you create an empty repository on GitHub (no README/license if you’re pushing this history first), run:

```bash
cd path/to/Malkshook
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

Replace `YOUR_USERNAME` and `YOUR_REPO` with your account and repository name.
